# Pizza Order App

A comprehensive pizza ordering application with user authentication, responsive design, and order management.

## Features

1. **User Authentication**
   - Sign up with email/password
   - Login with email/password
   - Simulated Google login
   - Session management

2. **Pizza Ordering**
   - Customize pizza size, crust, sauce, cheese, and toppings
   - Real-time order summary with price calculation
   - Quantity selection
   - Order placement

3. **Responsive Design**
   - Mobile-friendly navigation
   - Adaptive layout for all screen sizes
   - Professional styling

4. **Database Simulation**
   - User data storage (localStorage)
   - Order history
   - Pizza menu configuration
